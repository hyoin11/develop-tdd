# 블로그 서비스
- 주요 기능
  - 회원가입 / 로그인 (JWT 기반)
  - 게시글 crud (작성 / 수정 / 삭제 / 조회)
  - 댓글, 좋아요, 조회수 기능
  - 태그 기반 분류
- 확장 기능
  - Elasticsearch
    - 제목 / 본문 / 태그 통합 검색
    - 자동완성 (검색어 추천)
    - 인기 검색어 랭킹 (Redis + es 연동)
  - Redis
    - 조회수 캐싱 (조회수 집계 -> 일정 주기마다 db 반영)
    - 인기글 캐싱 (최근 24시 / 일주일 기준)
    - 세션 / 토큰 캐싱 (Spring security 연동)
  - 배치 작업 (spring batch)
    - 매일 / 매주 통계 집계 (예. 주간 인기글, 활동 많은 사용자)
  - API 문서화
    - swagger
  - 테스트
    - junit + testContainers (mysql, redis, es 통합 테스트)
- TDD 적용 방식
  - 도메인 로직이 중요한 부분부터 적용
  - 테스트 우선 적용 영역
    - 회원가입/로그인 (비밀번호 암호화, jwt 발급/겁증 등)
    - 게시글 crud (검증 로직, 권한 체크)
    - 조회수/좋아요 로직 (redis 캐시 + 동기화)
    - 검색 api (es 연동 전, mocking으로 먼저 테스트)
  - 도구
    - junit5 + assertJ (단위 테스트)
    - mockito (의존성 mocking)
    - spring boot test (통합 테스트)
    - testContainers (mysql, resid, es를 띄워 진짜 환경 테스트)
- 사용 기술
  - 백엔드
    - spring boot 3
    - spring data jpa (mysql)
    - spring security + jwt
    - elasticsearch (검색, 자동완성)
    - redis (캐시, 랭킹, 세션)
    - spring batch
    - junit + assertJ + testContainers
  - 인프라
    - aws
    - github actions
  - 프론트 (외부 템플릿 사용)
    - 블로그 템플릿 사용
    - 백엔드 rest api 연동만
