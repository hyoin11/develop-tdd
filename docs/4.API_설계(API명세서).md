# API 설계
### API 목록
##### 회원 
[회원 API 문서](./apis/users.md)
- 회원가입 (POST /api/v1/users)
- 회원 정보 조회 (GET /api/v1/users/me)
- 회원 정보 수정 (PATCH /api/v1/users/me)
- 회원 탈퇴 (DELETE /api/v1/users/me)
##### 인증/인가
[인증/인가 API 문서](./apis/auth.md)
- 로그인 (POST /api/v1/auth/login)
- 로그아웃 (POST /api/v1/auth/logout)
##### 게시글
[게시글 API 문서](./apis/posts.md)
- 게시글 목록 조회 (GET /api/v1/posts)
- 게시글 검색 (GET /api/v1/posts?query=...)
- 게시글 작성 (POST /api/v1/posts)
- 제시글 수정 (PATCH /api/v1/posts/{postId})
- 게시글 삭제 (DELETE /api/v1/posts/{postId})
- 게시글 상세 (GET /api/v1/posts/{postId})
- 인기글 조회 (GET /api/v1/posts/popular)
##### 댓글
[댓글 API 목록](./apis/comments.md)
- 댓글 목록 (GET /api/v1/posts/{postId}/comments)
- 댓글 작성 (POST /api/v1/posts/{postId}/comments)
- 댓글 수정 (PATCH /api/v1/comments/{commentId})
- 댓글 삭제 (DELETE /api/v1/comments/{commentId})
##### 좋아요
[좋아요 API 목록](./apis/likes.md)
- 게시글 좋아요 (POST /api/v1/posts/{postId}/like)
- 게시글 좋아요 취소 (DELETE /api/v1/posts/{postId}/like)
